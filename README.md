# MauroTools

## Entorno virtual (Windows / PowerShell)

El entorno virtual ya está creado en `.venv`.

Para activarlo:

```powershell
.\.venv\Scripts\Activate.ps1
```

Para desactivarlo:

```powershell
deactivate
```

Para instalar dependencias (si existe `requirements.txt`):

```powershell
python -m pip install -r requirements.txt
```
